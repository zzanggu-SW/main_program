## 프로젝트 설명

프로젝트 이름은 **Production Installer**. 이 프로젝트는 자사의 소프트웨어를 설치하기 위한 프로그램입니다.

### 주요 기능
- 선별기에 맞는 설정과 소프트웨어를 다운 받는 프로그램

## 설치 방법
- Windows에서 run.bat 실행하여 설치 및 실행

### 요구 사항

이 프로젝트를 실행하기 위해서는 다음이 필요합니다:
- Python 3.11 이상
- [Poetry](https://python-poetry.org/) 패키지 관리자
